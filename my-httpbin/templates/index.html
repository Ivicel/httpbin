httpbin(1): HTTP Request & Response Service

Free hosted in HTTP, HTTPS & EU flavors.

ENDPOINTS

/    This page.

/ip    Return Origin IP.

/user-agent    Returns user-agent.

/get    Returns GET data.

/post    Returns POST data

/patch    Returns PATCH data.

/put    Returns PUT data.

/delete    Returns DELETE data.

/encoding/utf8    Returns page containing UTF-8 data.

/gzip    Returns gzip-encoded data.

/deflate    Returns delate-encoded data.

/status/:code    Returns given HTTP Status code.

/response-headers?key=val    Returns given response headers.

/redirect/:n    302 Redirects n times

/redirect-to?url=foo    302 Redirects to foo URL.

/relative-rediret/:n    302 Relative redirects n times.

/absolute-redirect/:n    302 Absolute redirects n times.

/cookies    Returns cookie data.

/cookies/set?name=value    Sets one or more simple cookies.

/cookies/delete?name    Deletes one or more simple cookies.

/basic-auth/:user/:passwd    Challenges HTTPBasic Auth.

/hidden-basic-auth/:user/:passwd    404'd BasicAuth.

/digest-auth/:qop/:user/:passwd    Challenges HTTP Digest Auth.

/stream/:n    Streams min(n, 100) lines.

/delay/:n    Delays responding for min(n, 10) seconds.

/drip?numbytes=n&duration=s&delay=s&code=code

/range/1024?duration=s&chunk_size=code    Streams n bytes and allows specifying a Range header to select a subset of the data. Accepts a chunk_size and request parameter.

/html    Renders an HTML Page.

/robots.txt    Returns some robots.txt rules.

/deny    Denied by robots.txt file.

/cache    Returns 200 unless an If-Modified-Since or If-None-Match header is provided, when it returns a 304.

/cache/:n    Set a Cache-Control header for n seconds.

/bytes/:n    Generates n random bytes of binary data, accepts optional seed integer parameter.

/stream-bytes/:n    Streams n random bytes of binary data, accepts optional seed integer parameter.

/links/:n    Returns page containing n HTML links.

/image    Returns page containing an image based on sent Accept header.

/image/png    Returns page containing a PNG image.

/image/jpeg    Returns page containing a JPEG image.

/image/webp    Returns page containing a WEBP image.

/image/svg    Returns page containing a SVG image.

/forms/post    HTML form that submits to /post.

/xml    Returns some XML

DESCRIPTION

Testing an HTTP Library can become difficult somtimes. RequestBin is fantastic for testing POST request, but doesn't let you control the response. This exists to cover all kinds of HTTP scenarios. Additional endpoints are being considered.

All endpoint responses are JSON-encoded.

Installing and running form PyPI

You can install httpbin as a library from PyPI and run it as a WSGI app. For example, using Gunicorn: 

	$ pip install httpbin
	$ gunicorn httpn:app
